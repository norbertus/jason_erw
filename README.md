# Reise42 Angular-Example

# Demo for using APIs from DB MarketPlace

## Checkout and Install

```
git clone git clone https://github.com/norbertus/jason_erw


cd jason_erw
npm install
```
## Start Fake DB Json Server
cd backend
json-server --watch database.json

## Start

```
cd jason_erw
ng serve -o
```
